## 스레드 1: 배당주로 월급 만들기 — 현실적인 로드맵 🧵

---

**트윗 1 — 훅 (Hook)**
```
배당주로 월급 만들기, 실제로 가능할까요?

"배당으로 생활비 낸다"는 분들 실제로 있습니다.
근데 진짜 현실적인 로드맵은 이렇게 생겼어요.

10개 트윗에 정리했습니다 🧵
```

---

**트윗 2 — 현실 파악**
```
2/ 먼저 팩트 체크

배당수익률 4% 기준으로
월 100만원 배당 받으려면?

→ 원금 3억원 필요

처음 들으면 막막하죠.
근데 이걸 단번에 만들려고 하면 안 돼요.

10~15년 복리로 쌓아가는 겁니다.
```

---

**트윗 3 — 단계별 목표**
```
3/ 현실적인 단계별 목표

1단계 (1-3년)
→ 월 배당 1~3만원 목표
→ "배당이 이렇게 들어오는구나" 경험

2단계 (3-7년)
→ 월 배당 10~20만원
→ 통신비나 식비 일부 커버

3단계 (7-15년)
→ 월 배당 50만원 이상
→ 생활비 일부 대체

숫자보다 습관이 먼저입니다.
```

---

**트윗 4 — 배당주 선택 기준**
```
4/ 배당주 고르는 3가지 기준

① 배당 지속성
→ 5년 이상 배당 유지한 종목

② 배당성향 적정 여부
→ 50~70% 적정 (100% 이상은 위험)

③ 재무 건전성
→ 부채비율 확인 필수

배당수익률만 보고 고르면 함정에 빠집니다.
고배당이 항상 좋은 게 아님!
```

---

**트윗 5 — ETF vs 개별주**
```
5/ ETF vs 개별 배당주

ETF 장점:
→ 분산 자동화
→ 리서치 시간 절약
→ 초보에게 적합

개별주 장점:
→ 배당 성장 기업 집중 투자 가능
→ 기업 성장 수혜 직접 수령

내 추천: ETF 60% + 개별주 40%
초보라면 ETF 80~100%로 시작
```

---

**트윗 6 — 배당 재투자 복리**
```
6/ 배당 재투자가 핵심입니다

월 30만원 투자, 배당수익률 4%, 주가 연 5% 성장 가정

재투자 안 하면:
→ 10년 후 원금 3,600만 + α

재투자 하면:
→ 10년 후 약 5,800만원 이상

같은 돈, 같은 기간인데
1,600만원 이상 차이 납니다.
배당금은 무조건 재투자예요.
```

---

**트윗 7 — 세금 최적화**
```
7/ 세금도 챙겨야 진짜 수익

배당소득세 기본 15.4%
→ 연 2,000만원 초과 시 종합소득 합산

절세 전략:
① ISA 계좌 활용 (200만원 비과세)
② 연금저축에 배당 ETF 편입
③ 부부 계좌 분산 (각자 2,000만원 한도)

세금 고려 안 한 배당투자는
반쪽짜리입니다.
```

---

**트윗 8 — 흔한 실수**
```
8/ 배당투자 초보가 자주 하는 실수

❌ 배당수익률만 보고 매수
→ 주가 하락 시 총수익률 마이너스

❌ 배당금을 생활비로 사용
→ 초기엔 무조건 재투자해야 복리 효과

❌ 분기마다 종목 교체
→ 배당투자는 장기 보유가 핵심

❌ 한 종목에 집중
→ 배당 컷 한 방에 포트폴리오 타격
```

---

**트윗 9 — 지금 당장 할 것**
```
9/ 오늘 바로 시작하는 법

① ISA 계좌 개설 (10분, 무료)
② 배당 ETF 1개 선택
   (KODEX 배당성장 or TIGER 미국배당다우존스)
③ 월 자동이체 5만원이라도 설정
④ 배당금 들어오면 바로 재투자

완벽한 포트폴리오보다
시작하는 것이 훨씬 중요합니다.
```

## 스레드 2: ETF 수수료의 진실, 모르면 매년 돈 새는 중 🧵

**총 8개 트윗**

---

**트윗 1 — 훅 (Hook)**
```
ETF 투자하는데 수수료 확인 안 하시나요?

연 0.5% 차이가
10년에 수백만 원 차이 납니다.

진짜입니다.
계산해드릴게요 🧵
```

---

**트윗 2 — TER이 뭔가요?**
```
2/ ETF 수수료 = TER (총보수)

Total Expense Ratio

1,000만원 투자, TER 0.5% → 연 5만원 비용
1,000만원 투자, TER 0.07% → 연 7,000원 비용

차이: 연 43,000원

별거 아닌 것 같죠?
```

---

**트윗 3 — 복리로 보면 달라진다**
```
3/ 10년으로 늘려보면

월 30만원, 연 7% 수익률 가정

TER 0.07% ETF → 10년 후 약 5,220만원
TER 1.5% ETF → 10년 후 약 4,820만원

같은 돈, 같은 기간인데
약 400만원 차이!

수수료는 복리로 갉아먹습니다.
```

---

**트윗 4 — 국내 vs 해외 ETF 비교**
```
4/ 국내 ETF TER 비교 (2026년 기준)

나스닥 100 추종:
TIGER 미국나스닥100    0.07%
KODEX 미국나스닥100    0.05%
KBSTAR 미국나스닥100   0.02%

S&P 500 추종:
TIGER S&P500          0.07%
KBSTAR S&P500         0.02%

⚠️ 비슷해 보이지만 3배 이상 차이!
```

---

**트윗 5 — 숨겨진 비용들**
```
5/ TER 외에 숨겨진 비용들

① 매매 스프레드
→ 호가창 상/하단 차이
→ 거래량 많은 ETF 선택 필수

② 추적오차 (Tracking Error)
→ 지수와 ETF 수익률의 차이
→ 낮을수록 좋음

③ 거래세
→ 국내 ETF: 주식처럼 취급 없음
→ 채권형 ETF: 이자소득 과세

TER만 보면 반만 아는 겁니다.
```

---

**트윗 6 — ETF 선택 체크리스트**
```
6/ 수수료 싼 ETF 고르는 체크리스트

✅ TER 0.1% 이하 목표
✅ 순자산 1,000억 이상 (규모가 작으면 폐지 위험)
✅ 일평균 거래량 1억원 이상
✅ 설정일 3년 이상
✅ 추적오차 0.5% 이내

이 5가지 통과하면 합격입니다.
```

---

**트윗 7 — 비싼 ETF가 나쁜 건 아니다**
```
7/ 수수료 높은 ETF가 항상 나쁜 건 아닙니다

테마형 ETF (AI, 바이오, 우주항공 등)
→ TER 0.5~0.8%도 있음
→ 해당 테마 독점적 접근 가능

단, 같은 지수 추종 ETF라면
무조건 수수료 낮은 걸 선택

비교 가능할 때만 수수료 체크가 의미있음!
```

---

**트윗 8 — CTA**
```
8/ 마무리 + 행동 촉구

지금 당장 할 것:
→ 보유 ETF 수수료 확인
→ 동일 지수 추종 중 가장 낮은 TER 비교
→ 필요하면 갈아타기 검토

ETF 교체 시 세금 주의!
(차익에 배당소득세 15.4%)

이 스레드 도움 되셨다면 리트윗 부탁해요 🔁
질문은 댓글로!
```

---

**트윗 2 — 직장인 종합소득세 대상**
```
2/ 직장인이 종합소득세 내는 경우

① 금융소득 연 2,000만원 초과
   (이자 + 배당 합산)

② 부업/프리랜서 소득 있을 때
   (유튜브 수익, 강의료, 원고료 등)

③ 임대소득 2,000만원 초과

④ 근로소득 외 기타소득 300만원 초과

해당되면 5월에 반드시 신고해야 합니다.
```

---

**트윗 3 — 금융소득 종합과세 포인트**
```
3/ 금융소득 2,000만원 기준

배당 + 이자 합산 2,000만원 초과 시
→ 근로소득과 합산되어 종합과세

문제: 근로소득이 높은 직장인일수록
금융소득에 적용되는 세율이 높아짐

예: 연봉 7,000만원 + 금융소득 3,000만원
→ 일부 금융소득에 38% 세율 적용 가능
```

**트윗 4 — 절세 전략 1: ISA 계좌**
```

2,000만원 한도에 포함되지 않음!

연간 납입 한도: 2,000만원
비과세 한도: 200만원 (서민형 400만원)
초과분: 9.9% 분리과세

금융소득 분리 + 저율과세
이중 효과 있습니다.
```

**트윗 5 — 절세 전략 2: 연금계좌**

```
세액공제율: 13.2~16.5%

공제 후 절세액:
→ 총급여 5,500만원 이하: 최대 148만원
→ 총급여 5,500만원 초과: 최대 118만원

단, 연금으로 수령해야 혜택 유지
(중도 해지 시 패널티 주의)
```

**트윗 6 — 절세 전략 3: 손익통산**
```

12월에 손실 종목 정리

B를 12월에 매도 → 손실 확정

해외주식은 별도 양도소득세 과세
```
