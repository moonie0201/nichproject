```
❌ 한 기업이 망하면 큰 손실
```

**슬라이드 3 — ETF란?**
```
📌 ETF란?

여러 주식을 묶은 "바구니"

코스피200 ETF 1주 =
200개 기업에 동시 투자

✔ 분산투자로 리스크 줄임
✔ 주식처럼 실시간 거래 가능
✔ 수수료 저렴 (연 0.1~0.5%)
❌ 개별 종목만큼 큰 수익은 어려움
```

```

돈 모이는 습관
```

**슬라이드 2 — 왜 루틴이 필요한가?**
```
💬 "투자는 하고 싶은데
    시간도 없고 어디서부터
    시작해야 할지 모르겠어요"

이런 분들을 위해 준비했어요.

재테크는 대단한 게 아닙니다.
작은 습관이 쌓여서 만들어지는 것!

핵심은 '자동화'와 '루틴화'
```

**슬라이드 3 — 1단계: 월급날 루틴**
```
📅 1단계: 월급날 (D-Day)

월급 들어오면 10분 안에:

✅ 비상금 계좌로 자동이체
   (목표: 월 생활비 3개월치)
✅ 투자 계좌로 자동이체 설정
   (월급의 20~30%)
✅ 생활비 계좌 잔액 확인

"선저축 후소비"가 황금 원칙 🥇
먼저 빼놓지 않으면 다 써요!
```

**슬라이드 4 — 2단계: 주간 루틴**
```
📅 2단계: 매주 (5분이면 충분)

매주 금요일 저녁:
✅ 이번 주 지출 확인
   (가계부 앱 1분 체크)
✅ 투자 계좌 수익률 확인
   (주가 등락에 일희일비 금지)
✅ 다음 주 이슈 체크
   (경제 뉴스 헤드라인 1개)

⏱️ 전체 5분이면 충분합니다
```

**슬라이드 5 — 3단계: 월간 루틴**
```
📅 3단계: 매월 1회 (30분)

매월 1일:
✅ ETF 정기 매수 실행
✅ 순자산 변화 기록
   (자산 - 부채 = 순자산)
✅ 지난달 지출 카테고리 분석
✅ 다음 달 큰 지출 미리 계획

순자산이 늘어나는 걸 보면
재테크가 즐거워집니다 📈
```

**슬라이드 6 — 4-5단계: 분기·연간 루틴**
```

✅ 포트폴리오 리밸런싱
✅ 배당락일 전 배당주 점검
✅ 보험료 적정성 검토

📅 5단계: 연간 (반나절)

✅ 연말정산 세액공제 최적화
✅ ISA/연금계좌 납입 한도 확인
✅ 내년도 투자 목표 설정
✅ 순자산 연간 성과 정리
```

```
🛠️ 지속 가능한 루틴의 핵심
```
