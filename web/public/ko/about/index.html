InvestIQs Research#

InvestIQs는 미국 ETF · 한국 ETF · 배당 자산 · 장기 자산배분을 다루는 데이터 기반 리서치 플랫폼입니다. 모든 글은 공개 데이터와 AI 분석을 결합해 작성되며, 2단계 검증을 거쳐 발행됩니다.

분석 방법론#

  1. 실시간 데이터 검증

    • yfinance 가격·배당·기간수익률을 매일 캐시 갱신 (매일 08시 KST).
    • 배당수익률은 yfinance.info 값과 Ticker.dividends 직접 계산값을 교차 검증 — 30% 이상 차이 시 보수적 값 채택.
    • 이상치 자동 거부 (P/E 음수, 배당률 >15% 등 sanity 범위 벗어난 데이터는 버림).

  2. AI 멀티에이전트 분석

    • virattt/ai-hedge-fund 기반 5개 에이전트: 워렌 버핏 관점 / 기술적 / 펀더멘털 / 뉴스 심리 / 리스크 관리.
    • 각 에이전트 결과를 리서치 노트에 참고 자료로만 인용, 단일 신호로 결론 내지 않음.

  3. Peer ETF 비교

    • 단독 분석 지양 — 최소 1개 Peer와 운용보수·배당·총수익률 비교.

  4. Contrarian Angle + Disconfirming Evidence

    • 시장 컨센서스와 다른 관점 1개 이상 제시.
    • “이 분석이 틀릴 수 있는 시나리오” 최소 1건 명시.

  5. 2단계 검증

    • 1차: 규칙 검증 (글자수 / 비교표 / 금지어 / 원본 데이터 반영).
    • 2차: Gemini 의미 검증 (할루시네이션 / 내부 모순 / 규제 위반 표현 / 존재하지 않는 차트 참조).
    • 두 단계 모두 통과해야 발행.

운영 원칙#

  • 정보 제공 ≠ 투자 자문: 본 사이트의 모든 콘텐츠는 공개 데이터 기반 리서치 노트이며, 특정 종목의 매수·매도를 권유하지 않습니다.
  • 자본시장법 준수: 한국 독자 기준 금감원 규정 준수. “매수하세요”, “반드시 오른다” 같은 단정적 권유 표현 사용 금지.
  • 투명성: AI 보조 작성이 포함된 리서치임을 명시. 글 상단 규제 배너와 하단 면책 조항으로 안내.
  • 일일 발행: 매일 06~08시 KST 자동 발행 (5개 언어 병행).
