## Executive Summary (요약)

글로벌 투자 콘텐츠 시장은 **(1) 전문 저널리즘(Bloomberg, Barron's)** → **(2) 크리에이터 미디어(Substack, YouTube)** → **(3) AI 큐레이션(Finimize, Readless)** 의 3단 구조로 재편되고 있다.

핵심 관찰:
1. **Substack 기반 1인 애널리스트의 경제적 영향력**이 전통 매체를 위협하는 수준. Matt Levine의 *Money Stuff*는 Bloomberg 내에서만 **30만+ 구독자**, Ben Thompson의 *Stratechery*는 **40,000명 × $12/월 = 월 $48만 매출** 규모로 추정.
2. **YouTube 투자 크리에이터**는 개인 브랜드 + 라이프스타일 결합 형태가 우세 (Graham Stephan **4.3M**, Meet Kevin **1.9M**, Ramit Sethi **1.07M**).
3. **콘텐츠 포맷의 대변화**: 단순 "뉴스 전달" 보다 **짧은 해석(5분)** + **인터랙티브 차트** + **뉴스레터↔팟캐스트↔YouTube 크로스 퍼블리싱**.
4. **트렌드 주제**: AI 반도체, Fed 인하, 금·원유 지정학, 배당성장, 채권, 크립토 ETF가 상위 키워드.
5. **한국 ↔ 글로벌 gap**: 한국은 "개별 종목 단타/미디어 정보성" 편중, 글로벌은 "프레임워크 + 에듀케이션 + 리서치" 편중. 이 gap 자체가 기회.

### 8-B. 한국 → 영어 콘텐츠 확장 시 고려 사항

1. **언어 톤**: 한국어 직역 금지. 영어권은 **1문장 = 1 아이디어, 짧은 문단** 기본.
2. **인용/출처**: 미국권은 학술/데이터 인용 필수 (CRSP, FRED, SSRN). "내 느낌"만으로는 신뢰 0.
3. **규제 디스클레이머**: "Not financial advice", "I am not a financial advisor" 문구 표준.
4. **제목**: 낚시성 지양, **숫자 + 고유명사 + 구체 질문** 형식. ("3 Dividend ETFs That Beat SCHD in 2025")
5. **시각화**: FRED, Stockanalysis.com, TradingView 임베드 표준.

## 9. 영어 콘텐츠 템플릿 3종 제안

### Template 1: "Data-Driven Essay" (A Wealth of Common Sense / Of Dollars And Data 스타일)

**분량**: 700-1,200 words
**구조**:
```
[제목] A specific, non-clickbait question
ex) "What Happens to Stocks When the Fed Cuts Rates in an Expansion?"

[Hook] 1-2 paragraphs - 역사적 데이터나 반직관적 통계 1개 제시
[Chart] 1 chart (must be clean, single-insight, watermarked)
[Analysis] 3-4 paragraphs - 데이터를 "What does this mean?" 관점에서 해석
[Counterargument] 1 paragraph - 반론 또는 한계
[Conclusion] 2-3 sentences - "The bottom line is..."
[Disclosure] "I am long XYZ. Not financial advice."
```
**SEO**: H2/H3로 FAQ 키워드 자연 노출. Meta description 155자, "2026" 연도 포함.
**CTA**: 뉴스레터 구독 (top 10% conversion on this template).

### Template 2: "Comparative Review (Affiliate-Primed)" (Motley Fool / NerdWallet 스타일)

**분량**: 2,000-3,500 words
**구조**:
```
[제목] "X vs Y vs Z: Which [Product] Is Best in 2026?"
ex) "SCHD vs DGRO vs VIG: Best Dividend Growth ETF for 2026"

[Intro] 200자 - "공통점/차이점/결론 선공개"
[At-a-Glance Table] 10-row spec table
[Deep Dive #1: X] 
  - Overview, top holdings, expense ratio, 5-yr return, pros/cons
[Deep Dive #2: Y] 동일 구조
[Deep Dive #3: Z] 동일 구조
[Head-to-Head by Use Case]
  - "Best for dividend growth" → X
  - "Best for total return" → Y
  - "Best for low fees" → Z
[Our Pick] 명확한 winner 1개 + 이유 3개
[FAQ] 5-7 long-tail Q&A (AI Overview 타겟)
[Affiliate Link] 브로커 referral, expense ratio calculator 링크
```
**수익화**: M1 Finance/Interactive Brokers/Webull 어필리에이트.
**SEO**: "vs" 키워드, 비교 테이블이 Google SERP의 rich snippet 잡기 유리.

### Template 3: "Weekly Market Wrap" (The Compound / Abnormal Returns 스타일)

**분량**: 1,000-1,500 words (뉴스레터) + 15-20분 팟캐스트/유튜브
**구조**:
```
[Subject Line] "[Week XX] Three Charts That Mattered" (open rate 최적화)

[This Week's 3 Charts] 각 차트 200자 해석
  Chart 1: Macro (Fed, CPI, GDP)
  Chart 2: Sector Rotation (AI/energy/healthcare)
  Chart 3: Investor Behavior (flows, sentiment)

[Quick Hits - 5 Links] 링크 + 1줄 해석 (Abnormal Returns 스타일)

[Deep Dive of the Week] 1개 주제 300자 심층

[Tickers on My Radar] 3개 종목/ETF 간단 노트 (affiliate 없음, 신뢰 확보)

[Podcast/Video CTA] 이번 주 영상 썸네일 임베드

[Disclosure + Sub CTA]
```
**크로스 배포**: 동일 스크립트 → 뉴스레터 텍스트, 15분 팟캐스트 오디오, 20분 YouTube 영상 3중 배포.
**빈도**: 주 1회 (일요일 오전 발송 open rate 최고).

## 10. Actionable Insights (10+)

### 전략 레벨

1. **[Flywheel 구축] Ritholtz 모델을 벤치마크**: 블로그 (SEO/롱폼) → 뉴스레터 (오디언스 소유) → 팟캐스트 (권위) → YouTube (스케일) → 유료 커뮤니티/강의 (수익). 한국에서는 "유료 커뮤니티 → 강의" 순서로 역설계 권장.

2. **[포지셔닝] "Evidence-Based Investing" 영어 브랜드 선점**: 한국어 콘텐츠는 종목 중심 → 영어로는 **Ben Felix + Plain Bagel 밴드** (증거 기반, 장기투자, CFA 수준) 포지셔닝이 희소가치 보존 전략.

3. **[컨텐츠 해자] "Korean Market Translator for Global Investors"**: 영어권 독자에게 한국 반도체/2차전지/K-Content 기업의 인사이드 view 제공 → 독자 고유 (unique reader segment) 확보. KED Global 모델의 개인 크리에이터 버전.

### 실행 레벨

4. **[Substack 시작]** Beehiiv 또는 Substack 중 **Beehiiv 추천** (광고/레퍼럴 네이티브, 어필리에이트 친화). 출시 후 3-6개월은 무료로만 운영.

5. **[발송 빈도] 주 2회 수요일/일요일 고정**: 평일 1회 (마켓 분석) + 주말 1회 (에세이). 주 4회 이상은 번아웃/이탈.

6. **[장르 믹스] 포트폴리오 식 콘텐츠 운영**: 70% evergreen 교육 (SEO 수익) + 20% 시황/뉴스 (바이럴 트래픽) + 10% 개인적 포트폴리오/실험 (팬덤).

7. **[SEO 전술] "X vs Y" 형식으로 20개 비교 글 집중 발행**: 6개월 내 어필리에이트 MRR 셋업 가능. 예: `SCHD vs VIG`, `VOO vs QQQ`, `Webull vs Robinhood`.

8. **[AI 차별화] 한국어+영어 듀얼 퍼블리싱 자동화**: 원본 1개 → AI로 한국어/영어 동시 배포. 같은 주제도 한국어는 "전망/시황", 영어는 "framework/methodology" 톤으로 리라이팅.

9. **[인터랙티브 차트 도입]** TradingView Widget, Highcharts, Observable Plot 무료 임베드. 한국 재테크 콘텐츠 중 인터랙티브 차트 채택률 <5%, 글로벌 Top 20은 ~40%. 즉시 차별화 가능.

10. **[크로스 포맷]** 글 1편 = 팟캐스트 1편 (Descript로 자동 텍스트→오디오) = YouTube 1편 (HeyGen/D-ID 아바타). 초기 제작비 0원 확장.

11. **[수익화 단계적]** Phase 1 (0-6개월): 어필리에이트 only (Webull/M1/Simply Wall St). Phase 2 (6-12개월): 스폰서십 추가. Phase 3 (12개월+): 유료 뉴스레터 티어 $8/월.

12. **[신뢰 장치]** 모든 글에 **(1) 정확한 출처 링크 (2) 내 포지션 디스클로저 (3) "not financial advice" 문구**. 장기 브랜드의 85%는 "신뢰"로 결정됨.

13. **[커뮤니티]** Discord 서버 무료 런칭 + Patreon 유료 티어. 한국 독자는 카카오 오픈챗, 글로벌은 Discord 기본.

14. **[연구/리포트 패턴]** 분기 1회 "장문 리서치 리포트 (20페이지 PDF)" 발행 → 이메일 수집 + PR 효과. Doomberg Pro 모델 응용.

15. **[Observability]** Ahrefs/SEMrush 월 1회 경쟁사 5곳 (A Wealth of Common Sense, Of Dollars And Data, 등) traffic + keyword gap 분석 자동화.
