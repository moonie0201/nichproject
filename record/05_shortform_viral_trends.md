## 0. Executive Summary (TL;DR)

- **플랫폼 우선순위**: YouTube Shorts(수익성) > TikTok(도달률) > Reels(리치)
- **핵심 KPI**: 완시청률 70%+ (알고리즘 분기점), 1.7초 내 훅 결정

- **포맷 승자**: Reveal(결론 먼저) > Problem-Solution > List > Story
- **한국 최적 업로드**: 평일 오후 9시~자정 (레저 타임) + 오전 7~8시(출근길)
- **완시청률 벤치마크**: YouTube Shorts 5.9% 참여율, TikTok 3.15%, Reels 1.5%
- **주요 리스크**: SEC/FINRA/FCA가 finfluencer 규제 강화 중 → 면책/출처 표기 필수

### 1.2 플랫폼별 알고리즘 편애 특성 (2026 기준)

**YouTube Shorts**
- 롱폼 유입(구독 전환) 시 점수 급등 → "전체 영상 보러 가기" CTA 필수
- 검색 SEO 강점 (구글 연동) → 제목 키워드 중요
- Shorts 전용 수익화 조건: 구독자 500명 + 유효 공개 조회수 기반

**Instagram Reels**
- **Share가 왕**: WhatsApp/DM 공유가 가장 높은 가중치
- Save(저장) = "나중에 본다" 신호 → 정보성 콘텐츠 강점
- 좋아요는 2026년 가장 약한 신호로 강등

**TikTok**
- 2025년 말 알고리즘 3대 변화: Watch time + 검색 + 원본성
- 완시청률 70%+ × 1시간 이내 초기 참여 15%+ = 도달 3배
- AI 편집 탐지로 도용/리믹스 강등
- 70%의 투자 영상이 오도/과장 → 규제 리스크 상승

### 2.3 훅 공식 정리

1. **Curiosity Gap**: "X하는 1가지 방법" / "아무도 안 알려주는 Y"
2. **Results-First**: 결과 먼저 → "이렇게 했다" (63% CTR 상위 영상 패턴)
3. **Controversy**: 통념 깨기, 적 만들기
4. **POV/Story**: "제가 X년 전에..." — 감정 이입
5. **Numerical Specificity**: "3가지", "$1,000", "87%" — 구체 숫자가 일반화된 단어보다 2배 효과

## 3. 구조 템플릿 (포맷별)

### 3.1 Problem-Solution-CTA (15~45초, 범용)

```
[0~3초] 훅: 문제 제시 "주식 사면 바로 빠지는 이유"
[3~8초] 공감: "저도 그랬어요, 한 달에 -30%..."
[8~28초] 솔루션: 3가지 원칙 or 1가지 체크리스트
[28~40초] 증거/예시: 차트 or 실제 수익 인증
[40~45초] CTA: "프로필 링크에서 전체 영상"
```

- **적합 플랫폼**: 전 플랫폼
- **완시청률**: 62~71%
- **주의**: 공감 구간이 8초 넘으면 이탈 급증

### 3.2 Reveal Format (결론 선공개 → 이유)

```
[0~3초] 훅: 결론 선언 "2026년 무조건 사야 할 ETF: QQQM"
[3~6초] 의외성: "근데 이유가 충격적"
[6~30초] 이유 3가지 순차 설명
[30~35초] 반론 처리: "이거 걱정되실 텐데..."
[35~40초] CTA: "팔로우하고 2편 확인"
```

- **적합 플랫폼**: TikTok, Shorts
- **완시청률**: **최상 (75%+)**
- **Humphrey Yang, Mark Tilbury 대표 포맷**

### 3.3 List Format (TOP 3/5)

```
[0~3초] 훅: "배당주 TOP 5"
[3~8초] 1번: 종목 + 배당률 + 1줄 이유
[8~13초] 2번
[13~18초] 3번
[18~23초] 4번 (반전: "근데 이건 안 추천")
[23~28초] 5번: 베스트 픽
[28~30초] CTA
```

- **적합 플랫폼**: Shorts, Reels (Save 유도)
- **Save 비율**: 리스트 포맷이 다른 포맷 대비 3~5배
- **팁**: 마지막을 반전/베스트로 남기면 완시청률 +15%

### 3.4 Story Format (나의 실수/성공담)

```
[0~3초] 훅: "제가 25살에 3억 날린 이야기"
[3~10초] 상황 설정: 그 당시 내 상태
[10~25초] 사건/결정: 무엇을 했는가 (긴장감 고조)
[25~35초] 결과 + 교훈
[35~45초] CTA: "여러분은 이 실수 하지 마세요"
```

- **적합 플랫폼**: Reels (감정 공유), TikTok (Story arc)
- **DM/공유**: 스토리 포맷이 공유율 최상
- **주의**: 실제 인증 자료(계좌, 거래내역) 없으면 신뢰도 급락

### 3.5 Reaction/Duet Format (TikTok 특화)

```
[0~3초] 원본 영상 일부 + 본인 반응
[3~15초] "이거 사실인가?" 사실 체크
[15~30초] 본인 의견 + 데이터 반박
[30초] CTA
```

- **플랫폼**: TikTok Stitch/Duet, Reels Remix
- **알고리즘 부스트**: 원본 영상 트래픽 흡수

**[INSIGHT #5] 한국 최적 슬롯은 저녁 9시~자정 + 출근길 7~8시**
- 근거: PlayBoard/Naver 트래픽 피크, 한국 late-night 문화
- 적용: 메인 업로드 21:30 KST 자동 스케줄

**[INSIGHT #6] TikTok은 상업 계정 CML 필수, 최소 하루 1편**
- 근거: 2026 알고리즘 업데이트 — 빈도 하락 시 도달 급락
- 적용: Buffer/Hootsuite로 TikTok 일일 자동 업로드

**[INSIGHT #7] YouTube Shorts는 롱폼 유입 트리거로 사용**
- 근거: 숏폼 RPM $0.05~$0.30 vs 롱폼 $10~$25 (30~500배 차이)
- 적용: 모든 Shorts에 "전체 영상" CTA + 고정 댓글

**[INSIGHT #8] Finfluencer 규제 리스크 → 면책/출처 표기 필수**
- 근거: SEC, FINRA, FCA 모두 2025~2026 규제 강화
- 적용: "투자자문 아님, 개인 의견" 자막 + 출처 표기 자동화

**[INSIGHT #9] 얼굴 + 차트 PIP가 차트 단독보다 완시청률 +15%**
- 근거: 인간 얼굴이 스크롤 정지 신호
- 적용: 얼굴 노출 꺼리면 AI 아바타 대신 손/제스처 B-roll

**[INSIGHT #10] 리스트 포맷(TOP 3/5)은 Save율 3~5배**
- 근거: Instagram "저장하고 나중에 보기" 심리
- 적용: 주 1편 이상 리스트 포맷 고정

**[INSIGHT #11] 해시태그 3~5개가 골든 레인지**
- 근거: 10개+는 오히려 리치 25% 감소 (Instagram)
- 적용: 대형1 + 중형2 + 니치2 피라미드 공식

**[INSIGHT #12] 한국어/영어 듀얼 채널 전략이 RPM 극대화**
- 근거: 영어권 RPM 2~3배 + 글로벌 트래픽
- 적용: 동일 스크립트 영어 더빙 버전 병행 업로드 (ElevenLabs 활용)

**[INSIGHT #13] 원본성 AI 탐지 → 리믹스/도용 강등**
- 근거: TikTok 2026 AI 탐지 강화, Shorts도 도용 콘텐츠 필터링
- 적용: B-roll, 음원, 스크립트 최소 50% 오리지널 확보

**[INSIGHT #14] 자막은 단어 단위 Pop-up이 블록 자막보다 완시청 +8%**
- 근거: CapCut Auto Caption Word-level 벤치마크
- 적용: 전 영상 word-level 자막 자동화

**[INSIGHT #15] 썸네일 숫자 1개 + 얼굴 + 빨간 화살표 = CTR 상위 패턴**
- 근거: 월부, Humphrey Yang 등 상위 채널 공통 패턴
- 적용: Canva 템플릿 3종 고정 → 숫자/얼굴만 교체

**[INSIGHT #16] 첫 3초 "손실 회피" 훅이 "욕망" 훅보다 CTR +30%**
- 근거: 행동경제학 — 손실이 이득의 2배 강도 (Kahneman)
- 적용: "100만원 버는 법" 대신 "100만원 잃지 않는 법"

## 12. 자동화 파이프라인 연계 제안 (nichproject 연동)

### 12.1 기존 auto_publisher 확장

```
shorts/
├── scripts/           # 10개 템플릿 JSON 저장소
│   ├── reveal_kr.json
│   ├── list_kr.json
│   ├── story_kr.json
│   ├── reveal_en.json
│   └── list_en.json
├── assets/
│   ├── music/         # CML 라이선스 BGM
│   ├── broll/         # 차트/숫자 애니메이션
│   └── thumbnails/    # Canva 템플릿 PSD
├── generators/
│   ├── script_gen.py  # LLM 기반 스크립트 생성
│   ├── tts.py         # ElevenLabs 음성 합성
│   ├── video_gen.py   # MoviePy/FFmpeg 조립
│   └── caption.py     # Word-level 자막 생성
└── publishers/
    ├── youtube_shorts.py
    ├── instagram_reels.py
    └── tiktok.py
```

### 12.2 자동화 KPI 모니터링

- 완시청률 (Retention) 알림: 60% 미만 시 훅 A/B 테스트
- 도달률(Impressions/Follower): 주간 리뷰
- 팔로우 전환율: Shorts → 채널 구독 비율 추적
- RPM 비교: 스크립트 포맷별 수익 분석

### 12.3 단계별 로드맵

**Phase 1 (1~2주)**: 스크립트 템플릿 10개 JSON화, LLM 생성 파이프라인
**Phase 2 (2~4주)**: TTS + MoviePy 조립, 자막 자동화
**Phase 3 (4~6주)**: 멀티 플랫폼 자동 배포 (Shorts/Reels/TikTok)
**Phase 4 (6~8주)**: A/B 테스트 자동화 + 성과 피드백 루프

- **한국 금융소비자보호법**: 광고 공시 의무, 손실 가능성 고지

```
한국어:
"본 콘텐츠는 투자 권유가 아닌 개인 의견이며,
투자 결과에 대한 책임은 본인에게 있습니다."

English:
"Not financial advice. Do your own research. 
Past performance doesn't guarantee future results."
```

- 종목 추천 시 출처 3개 이상 교차 확인

- 실제 거래 인증은 모자이크 + 타임스탬프 필수
