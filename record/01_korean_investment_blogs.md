## Executive Summary (핵심 요약)

1. **월배당·커버드콜 ETF가 2026년 최강 핫토픽**: 국내 월배당 ETF 순자산 총액 약 58조9,300억원 돌파(2026.04.18 기준). 커버드콜 ETF가 신규 자금 유입 1순위.
2. **절세계좌 3종(ISA/연금저축/IRP)은 상시 고인기 에버그린 키워드**: 모든 상위 블로그의 핵심 트래픽 엔진.
3. **제목 패턴은 "연도 + 완벽정리/총정리 + 숫자" 3요소 반복**: 예) "2026년 월배당 ETF TOP 10 완벽 정리", "배당소득세 15.4% 아끼는 계좌 전략 — 2026 완벽 정리".
4. **글 길이 3,000자 이상이 SEO 상위 노출 기본선**: 구글 기준 1,500~3,000자, 네이버 블로그는 약 1,800~2,500자가 최적.
5. **플랫폼별 분업 구조가 정착**: 네이버 블로그(체험단·제휴) + 티스토리/워드프레스(애드센스·long-tail SEO) + 유튜브(브랜딩) + 인스타툰(인입).
6. **AI·반도체·원전·조선·방산 + 배당/월배당 + ISA 절세**가 2026년 상반기 3대 콘텐츠 축.
7. **직접적 수익화는 애드센스, 간접 수익화는 쿠팡파트너스·강의·전자책**이 정석.

## 3. 글 포맷·구조 분석

### 3.1 평균 지표 (관찰 기반)

| 항목 | 티스토리/워드프레스 | 네이버 블로그 |

| 평균 글 길이 | 3,000~5,000자 | 1,800~2,500자 |
| H2 개수 | 5~8개 | 3~5개 |
| H3 개수 | 10~20개 | 4~8개 |
| 표(Table) 사용 | 2~4개/글 | 0~2개/글 |
| 이미지 | 3~8장 | 5~15장 (네이버 가중치) |
| CTA 위치 | 글 하단 "함께 보면 좋은 글" | 프로필/이웃추가 |
| 내부 링크 | 3~5개 (관련 글) | 2~3개 |
| 외부 신뢰 소스 인용 | 한경·조선비즈·운용사 | 뉴스링크 복붙 |

### 3.2 정형화된 구조 (상위 노출 블로그 공통)

```
[도입부: 문제제기 + 결론 한줄 요약 (150~300자)]
[목차 Table of Contents]
[H2 1] 정의/개념 (초보 배려)
[H2 2] 현황/데이터 — 표 필수
[H2 3] 비교/순위 — 표 필수

[면책조항: 투자 권유 아님]
```

### 3.3 제목 반복 패턴 (SEO 클릭베이트)

**구조 공식**: `[숫자/연도] + [키워드] + [감성 트리거] + [범위/장르]`

자주 쓰이는 수식어:
- **연도형**: "2026년", "2026", "2026 완벽 정리"
- **완결성**: "완벽 정리", "총정리", "한방에", "A to Z", "끝판왕", "마스터 가이드"
- **숫자 리스트**: "TOP 10", "7가지", "5가지 전략", "3분 요약"
- **긴급/FOMO**: "지금 안 사면 평생 후회", "~몰랐다면 손해", "이것만 알면"
- **질문형**: "~해도 될까?", "~언제 팔아야 할까", "~어떻게 고를까"
- **수치 대비**: "15.4% 아끼는", "93만원 절세", "월 100만원"
- **타겟 명시**: "40대 직장인", "사회초년생", "주부", "초보자"

실제 상위 노출 제목 사례:

## 7. 우리 프로젝트 적용 Actionable Insights (15가지)

### A. 콘텐츠 전략

1. **월배당/커버드콜 카테고리 집중 신설**: 2026년 최강 트렌드. "KODEX 200타겟위클리커버드콜 분배금 추이", "반도체TOP10커버드콜 vs 나스닥100 커버드콜" 등 데이터 기반 포스트를 주 2회 자동 발행.
2. **ISA/연금저축/IRP 계좌 콘텐츠를 에버그린 기반(Pillar Content)으로 구축**: 각 주제당 3,000자 이상 "기둥 페이지" 1개 + 롱테일 주제 10개로 허브앤스포크 구조. 월 1회 연도 업데이트 자동화(제목 "2026 완벽 정리" → "2027 완벽 정리" 자동 교체).
3. **제목 템플릿화**: auto_publisher/content_generator.py에 `{연도} {키워드} {수식어}` 조합 룰을 하드코딩. 수식어 풀: ["완벽 정리", "총정리", "한방에", "끝판왕", "TOP N", "A to Z", "현실", "후회 없는"]. A/B 테스트 가능하게 설계.
4. **숫자·표 강제 삽입 규칙**: 포스트당 표 ≥2개, 숫자 제목 의무화(TOP 10/7가지/5가지). Hugo 숏코드로 ETF 분배율/PER/시총 표 자동 생성.
5. **초보/중급/고급 3단 타겟팅**: 각 포스트에 태그 `level:초보|중급|고급`을 붙이고, Hugo 리스트 페이지 & Twitter 해시태그(#재테크초보 #배당투자중급)에 반영.

### B. SEO/트래픽

6. **시즌 캘린더 자동화**: 연말정산(1월), 종합소득세(5월), 중간예납(11월), 종부세(6·12월)에 맞춰 auto_publisher에 날짜 트리거로 7~14일 전 집중 발행 큐를 설정.
7. **롱테일 키워드 시드 30종을 topic_manager.py에 심기**: 본 보고서 4장의 30+ 키워드를 seed로 등록, LLM이 변형·확장. 블랙키위·네이버 데이터랩 월간 체크로 상승 키워드 자동 승격.
8. **경쟁이 약한 다국어 크로스포스팅**: 한국 인기 주제(SCHD·JEPI·ISA)를 en/ja/vi/id로 현지화. 영어 버전은 "Korean investor's guide to ..." 포지션으로 dividend-nomad.com 외 공백 공략.
9. **글 길이 하한 3,000자 / 상한 6,000자 룰**: content_generator에서 최소 글자수 조건 충족 시에만 발행. 네이버 블로그 퍼블리싱 시에는 1,800~2,500자 버전으로 다운사이즈.

### C. 수익화

10. **쿠팡파트너스 링크 자동 삽입 파이프라인**: 재테크 도서(돈의 속성, 마법의 연금 굴리기 등) 또는 재무다이어리·가계부 상품을 글 유형별(초보 입문/세금/부동산)로 매핑한 링크북 YAML을 만들고 hugo.py publisher가 글 유형 태그에 따라 자동 삽입. 필수 면책 문구 자동 주입.
11. **증권사 CPA 제휴 배너 섹션** (토스증권·키움·미래에셋): 한국어 글 하단 고정 CTA 모듈을 Hugo partial로 만들고 배당/절세 주제에 노출.
12. **월 9,900원 한국어 프리미엄 뉴스레터 MVP**: 무료 공개 80% + 유료 20%(월간 ETF 분배율 예측표·세금 체크리스트). stibee 또는 maily 연동.

### D. 포맷/브랜딩

### E. 품질/신뢰

16. **권위 소스 자동 인용 규칙**: 한경·조선비즈·KB의 생각·미래에셋 매거진·운용사 리포트를 우선 인용. content_generator가 본문에 외부 링크 ≥3개 필수 삽입(nofollow 분리).
